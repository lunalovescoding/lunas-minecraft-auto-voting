Luna's Auto-Voting for Minecraft Servers

Automatic voting on voting websites for Minecraft to recieve ingame rewards.

## Features

- Automatic voting on 4 major server listing sites
- Project/Server management system
- Support for multiple servers
- Captcha detection warnings
- Configurable voting intervals
- Voting statistics and history
- Chrome and Edge compatible

## Supported Sites

- minecraft-mp.com
- minecraftservers.org
- minecraft-server-list.com
- minecraft-server.net

## Installation

### From Source

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" (top right corner)
4. Click "Load unpacked"
5. Select the extension folder

## Usage

1. Click the extension icon in your Chrome toolbar
2. Click "Add Project" tab
3. Enter your project details:
   - **Project Name**: Your server name (e.g., "My Minecraft Server")
   - **Username**: Your Minecraft username
   - **Rating URL**: The voting page URL (e.g., `https://minecraft-mp.com/server/40670/vote/`)
4. Click "Add Project"
5. The extension will automatically vote when you visit voting pages

### Tips

- You can add multiple servers/projects
- The extension detects captchas and will notify you
- Check the "Projects" tab to manage your servers
- Enable/disable projects individually

## Privacy

- All data is stored locally in Chrome's storage
- No data is sent to external servers
- Username and server info stay on your device
- Extension only operates on voting pages you add

## Troubleshooting

**Extension not voting automatically?**
- Make sure the project is enabled (check mark in Projects tab)
- Verify the voting URL is correct
- Check if the site has captcha protection

**Captcha detected?**
- Some sites require manual captcha solving
- The extension will notify you when captcha is needed
